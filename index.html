Página pessoal

Este site visa apenas o aprendizado pessoal sobre desenvolvimento web, portanto, isso explica o porquê está super simples. Estou criando a partir do zero para efeitos de estudo.

Até o presente momento este site está só o HTML5 puro. Depois irei implementar o CSS3, JavaScript e alguns frameworks.

Este site está hospedado no GitHub Pages que serve principalmente para prover páginas na internet sobre os repositórios do GitHub. Essas páginas podem servir para divulgar exemplos, demos, documentações e qualquer outro tipo de informação sobre o seu projeto. E o melhor, esse serviço é de graça, basta ter uma conta no GitHub.
